About This App

WeatherNow is a simple web app that helps users check current weather conditions, including temperature and wind speed, in any city worldwide.

It is built using modern web technologies such as:

  • HTML5 for structure
  • CSS3 with responsive design
  • JavaScript (ES6+) for interactivity and API access
  • OpenWeatherMap API for real-time weather data
  • LocalStorage for search history

This app demonstrates how clean design and core front-end technologies can be combined to build functional, user-friendly applications.
